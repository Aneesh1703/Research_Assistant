# research_assistant

Scaffold for a research assistant RAG-style project.

Structure created by scaffolding script. Fill in implementations for each module.

Quick start

1. Create and activate a virtualenv

   python -m venv .venv; .\.venv\Scripts\Activate.ps1

2. Install dependencies

   python -m pip install -r requirements.txt

3. Run the app (placeholder)

   python main.py
